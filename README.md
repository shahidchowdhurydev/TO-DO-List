# 📝 TO-DO List

A clean and minimal To-Do List web application built with HTML, CSS, and JavaScript.

---

## 🚀 Features

- Add new tasks
- Delete tasks
- Responsive and clean UI
- Beginner-friendly JavaScript project
- Simple task management system
